# Guía de Contribución

Para contribuir a este proyecto, por favor sigue estos pasos:

1.  Crea una rama (branch) desde `develop`.
2.  Asegúrate de seguir el estilo de código existente (Prettier/ESLint).
3.  Abre un Pull Request y espera aprobación antes de hacer merge.

¡Gracias por ayudar a construir Catalyst!
